# DevBrief
AI-powered tool built on IBM Bob that transforms complex codebases into plain English — so new developers can contribute from day one.

## What it does
- Connects to any GitHub repository
- IBM Bob scans the entire codebase
- Generates a plain English project summary
- Creates a file breakdown
- Produces a getting started guide for new developers

## Built with
- IBM Bob
